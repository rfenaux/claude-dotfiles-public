# Breeze AI Complete Guide

## Overview

Breeze is HubSpot's unified AI platform launched at INBOUND 2024 (September 18, 2024). It encompasses all AI features across the HubSpot platform.

## Breeze Components

### 1. Breeze Copilot (formerly ChatSpot)

**Availability:** All tiers (with limits)

**What it does:**
- AI companion built directly into HubSpot
- Accesses Smart CRM for personalized suggestions
- Helps with insights, content generation, CRM interaction

**Capabilities by tier:**

| Feature | Free/Starter | Professional | Enterprise |
|---------|--------------|--------------|------------|
| Basic chat | Yes | Yes | Yes |
| Content generation | Limited | Full | Full |
| Company research | No | Yes | Yes |
| Ticket summarization | No | Yes | Yes |
| Persistent memory | No | Yes | Yes |
| Custom assistants | No | No | Yes |

**Use cases:**
- Draft emails and content
- Research companies before meetings
- Summarize CRM records
- Get quick data insights
- Generate reports

---

### 2. Breeze Agents

#### Content Agent (Professional+)
**Purpose:** Generate multi-format content at scale

**Capabilities:**
- Blog posts from prompts or existing content
- Landing pages
- Case studies
- Podcasts (with AI voices)
- Social media posts
- Email campaigns

**Best practices:**
1. Train on brand voice using existing content
2. Provide clear prompts with context
3. Always review and edit output
4. Use for first drafts, not final copy

---

#### Prospecting Agent (Professional+)
**Purpose:** Research and personalize sales outreach

**Capabilities:**
- Identify high-quality leads
- Research target accounts
- Tailor outreach messages
- Send personalized sequences

**Integration:**
- Works with Sales Workspace
- Connects to LinkedIn Sales Navigator
- Uses Breeze Intelligence data

---

#### Customer Agent (Enterprise only)
**Purpose:** 24/7 AI-powered customer support

**Capabilities:**
- Handle basic queries using knowledge base
- Resolve 50%+ of support tickets automatically
- Smooth handoffs to human agents
- 40% reduction in ticket closing time

**Requirements:**
- Service Hub Enterprise
- Well-maintained knowledge base
- Defined escalation rules

**Setup:**
1. Configure knowledge base articles
2. Define response templates
3. Set escalation criteria
4. Train on common queries
5. Monitor and refine

---

#### Social Media Agent (Professional+)
**Purpose:** Optimize social media presence

**Capabilities:**
- Analyze performance metrics
- Study audience behavior
- Monitor industry trends
- Generate post recommendations
- Create social content

---

#### Data Agent (Professional+)
**Purpose:** Data enrichment and insights

**Capabilities:**
- Enrich contact/company data
- Identify data quality issues
- Surface patterns and trends
- Recommend automations

## Best Practices

### Content Generation:
1. Provide detailed context in prompts
2. Specify target audience
3. Include brand guidelines
4. Review all AI output
5. Edit for accuracy and tone

### Data Enrichment:
1. Set data quality rules first
2. Define enrichment priorities
3. Monitor credit usage
4. Verify enriched data periodically

### Agent Deployment:
1. Start with limited scope
2. Monitor performance metrics
3. Gather user feedback
4. Iterate and improve
5. Expand gradually

### Customer Agent:
1. Build comprehensive KB first
2. Define clear escalation paths
3. Monitor resolution rates
4. Review conversation logs
5. Update KB based on gaps

## Pricing & Credits

**Breeze Intelligence:**
- Credits-based system
- Included credits vary by tier
- Additional credits purchasable
- Usage visible in settings

**Breeze Agents:**
- Included in Professional+
- Enterprise required for Customer Agent
- No per-use charges

## Limitations

1. **Knowledge cutoff**: AI models have training data limits
2. **Accuracy**: Always verify factual claims
3. **Brand voice**: May need multiple iterations
4. **Complex queries**: May require human oversight
5. **Regional availability**: Some features geo-restricted
